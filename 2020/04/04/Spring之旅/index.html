Spring之旅

Spring是一个开源框架，Spring根本使命：简化Java开发。

为了简化java开发的复杂性，Spring使用如下4种策略：

1、基于POJO(Plain Old Java Object 简单Java对象 )的轻量级和最小侵入性编程。

2、通过依赖注入和面向接口实现松耦合。

3、基于切面和惯例进行声明式编程。

4、通过切面和模板减少样板式代码。
